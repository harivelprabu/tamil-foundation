# தமிழ் அடித்தளம் — Tamil Foundation

An interactive 4-week Tamil foundation programme for absolute beginners, built as a single self-contained web page. No installation, no server, no internet needed once loaded.

**▶️ Live site:** https://harivelprabu.github.io/tamil-foundation/

---

## What's inside

| Week | Focus | Lessons |
|------|-------|---------|
| 1 | **Uyir Ezhuthu** — the 12 vowels | 10 |
| 2 | **Mei Ezhuthu** — the 18 consonants | 10 |
| 3 | **Uyirmei & Numbers** — combined letters, 1–10 | 10 |
| 4 | **Words & Sentences** — colours, family, animals, food, first sentences | 10 |

Each week ends with a **test** of 12 multiple-choice questions, split into exam-style Sections A–D.

## Features

- **🔊 Audio reading** — every lesson, letter and word can be read aloud. Tamil letters use a Tamil voice, English explanations an English voice, via the browser's built-in speech engine.
- **📝 Exam-model tests** — all multiple-choice (no Tamil typing needed). Correct answers appear in green, mistakes in red, and **every wrong answer is explained**.
- **🎲 No guessable pattern** — the correct answer is spread evenly across positions A/B/C/D (12 each across the 48 questions), with never more than two in a row.
- **👩‍🏫 Teacher Access** — every submitted test is saved with a full answer sheet (question, chosen answer, correct answer, explanation). Includes CSV export and print.
- **↺ Reset Training** — clears progress for a fresh start, with the option to keep or clear teacher records.
- **📈 Progress tracking** — saved in the browser, so it remembers completed weeks.

## Usage

Open the site in a browser and work through one week at a time. Progress and results are stored locally in that browser (`localStorage`), so use the same device/browser to review a student's history under **Teacher Access**.

### Adding a Tamil voice for audio

If Tamil letters are read with an English accent, install a Tamil voice:

- **Windows** — Settings › Time & Language › Language › add **Tamil**
- **Mac / iPhone / iPad** — Settings › Accessibility › Spoken Content › Voices › add **Tamil**
- **Android** — Settings › Accessibility › Text-to-speech › install Tamil voice data

## Tech

A single `index.html` — plain HTML, CSS and JavaScript. No frameworks, no build step, no dependencies. Works offline.
